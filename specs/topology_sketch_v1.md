# Topology Sketch v1

- Online signed sketches track positive, negative, and cancellation channels.
- Minimal fields:
  - `positive_components`
  - `negative_components`
  - `cancellation_pairs`
  - `cycle_hint`
- Exact topology for this phase is delivered offline through `hm_std::exact_persistence`.
- Exact persistence output types:
  - `ExactPersistenceInput`
  - `BirthDeathPair`
  - `PersistenceDiagram`
  - `PersistenceSummary`
  - `OfflineTopologyReport`
- Python mirrors those Rust outputs with typed offline reports and JSON artifact export helpers.
- Primary research-artifact path:
  - trace JSON
  - persistence JSON
  - paired comparison JSON
- Metrics derived and carried into persistence summaries:
  - topological susceptibility
  - bridge dependence
  - tract retention
  - gluing defect
