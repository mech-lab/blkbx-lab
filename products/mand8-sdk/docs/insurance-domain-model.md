# MAND8 Insurance Domain Model

MAND8 keeps the shared receipt envelope but speaks in insurance-native terms.

## Core Objects

- Risk Receipt
- Exposure Unit
- Authority Receipt
- Control Receipt
- Incident Receipt
- Override Receipt

## UK-First Frame

Delegated authority, binder references, managing agents, coverholders, and FCA or PRA defensibility are first-class concepts in the MAND8 framing.

## Bundle Audiences

- Lloyd's underwriter
- managing agent
- MGA or coverholder
- reinsurer
- accelerator reviewer
