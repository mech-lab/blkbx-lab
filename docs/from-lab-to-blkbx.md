# From BLKBX Lab to BLKBX Commercial

BLKBX Lab is the open-source foundation for accountable AI agents. It provides the tools to generate local, verifiable Ink Receipts for agent actions.

## The Bridge

- **BLKBX Lab** creates local receipts. It is a developer tool for wrapping agents with action gates.
- **BLKBX Commercial** stores evidence ledgers and builds insurer-reviewable packets. It provides the infrastructure for enterprise-grade accountability and compliance.

**Important Note:** BLKBX Lab does not sell insurance and does not decide coverage. It provides the technical proof that an agent acted within its mandate and policy constraints.

## Future Export Capabilities

In a future release, BLKBX Lab will support exporting evidence bundles and receipts to the BLKBX Commercial platform:

```bash
blkbx-lab export --format blkbx-packet
```
