# Migration from mech-lab

BLKBX Lab is the evolution of `mech-lab`. The focus has shifted from mechanistic interpretability to accountable AI receipt gates.

## Package Name

- Old: `mechlab-sdk`
- New: `blkbx-lab`

## CLI

- Old: `mechlab`
- New: `blkbx-lab`

## Python Import

- Old: `import mech_lab`
- New: `import blkbx_lab`

## Artifacts

- `mair_manifest.v1.json` -> `ink_manifest.v1.json`
- `assurance_receipt.v1.json` -> `ink_receipt.v1.json`
- `backend_comparison.v1.json` -> `receipt_comparison.v1.json`
