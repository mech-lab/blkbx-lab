# Brand System

This directory is the release-facing brand source of truth for `mech-lab`.

Core truth:

- short tagline: `Mechanistic interpretability that ships.`
- technical strapline: `Architecture-agnostic IR · Sheaf holonomy · SLSA L3 provenance`
- voice: precise, direct, earned, grounded

Canonical files:

- [`tokens.json`](tokens.json): color, typography, spacing, and status semantics
- [`tokens.css`](tokens.css): docs/readme token implementation
- [`usage.md`](usage.md): short engineer guidance for release-facing copy and visuals
- [`release-copy.md`](release-copy.md): launch-ready copy blocks for README, PyPI, GitHub Releases, and social

Launch asset kit:

- [`../../assets/brand/og-card.svg`](../../assets/brand/og-card.svg): source-of-truth Open Graph card
- [`../../assets/brand/og-card.png`](../../assets/brand/og-card.png): GitHub social preview export for the live repo host
- [`../../assets/brand/launch-card.svg`](../../assets/brand/launch-card.svg): release announcement card
- [`../../assets/brand/release-header.svg`](../../assets/brand/release-header.svg): release note header
- [`../../assets/brand/badges/`](../../assets/brand/badges): proof/status badges
- [`../../assets/brand/icons/`](../../assets/brand/icons): launch icon subset

Naming rule:

- use only `mech-lab`, `mechlab`, and `mech_lab` in release-facing materials
- treat `HybridTDA`, `hybridtda`, and `pip install hybridtda` as outdated examples only
